# Gym tyme

# Description:
   Gym tyme is a responsive gym landing page, showcases gym equipment, gym accessories with a clean and modern layout.It also includes a booking form where users can enter their email, gender and choose the type of gym equipment they want to book.The form submission is handled using Formspark,allowing the details to be received via email without creating a backend.

# Features:
-Responsive gym landing page
-Displaying gym equipment
-Equipment booking form for users
-Form submissions integrated with Formspark
-Clean and structured UI layout

# Technologies Used:
HTML5
CSS
